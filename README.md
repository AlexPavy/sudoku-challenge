# sudoku-challenge
### Sudoku generator

Each time the page is refreshed, the server generates a new random Sudoku game with a unique solution

Uses :
* [Spring initializr](http://start.spring.io/)
* [Thymeleaf](http://www.thymeleaf.org/)
* [Angularjs](https://angularjs.org/)
